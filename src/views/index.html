fhirbase

Open-Source Platform for Health-IT applications based on FHIR standard

L

fhirbase is a transparent stack of independent components, services and libraries for fast development of HealthIT applications.


Front-End Applications

Front-End (SPA) applications could be created and distributed in a minutes using

  • fhir.js - generic javascript client library for FHIR-complaint REST-servers
  • fhirbase.js - npm module to publish

We've prepared templates & examples for AngularJS

  • AngularJS - npm module to publish
  • formstamp.js - npm module to publish

Application Layer

On application layer you could use any technology which could communicate with REST api or PostgreSQL. Possible scenarious - sending & receiving messages HL7 v2, CDA documents

  • fhirplace - thin REST-service for fhirbase
  • reference implementations - any FHIR reference implementations
  • Integration with fhirbase with SQL connection api (JDBC etc)

Start from getting started

Data Layer

On data layer we have PostgreSQL database with full-powered SQL and ACID. Also new jsonb datatype with jsquery (query language). Analysis, reporting, optimization and complex custom queries could be implemented on this layer.

  • fhirbase - generic javascript client library for FHIR-complaint REST-servers
  • jsquery - npm module to publish
